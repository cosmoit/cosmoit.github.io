Common Library for Application Workflows (CLAW)

CLAW is my engine/framework that I use to investigate new topics and implement features for applications that I frequently will use while either building a tool application or a game.

This include topics such as:

  • Rendering via OpenGL/Vulkan
  • Win32 apis
  • Input
  • C programming, memory management
  • UI systems
  • Font Rendering
  • GPU acceleration
  • Multithreading
  • Collision detection
  • Spatial acceleration structures
  • etc

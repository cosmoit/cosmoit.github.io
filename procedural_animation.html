Procedural Character Animation

I've done quite a bit of procedural animation work when it comes to characters as well as first person view models. Implementing things such as inverse kinematics and experimenting with different easing methods like springs solvers. The goal with character animation was to get something fluid with minimal effort in transitioning or conflict with other ongoing animation.

It was nice, once done, to have the ability to tune parameters like step size and speed in real time and have it affect all aspects of the character animation. Quickly and with little effort you can create some pretty good looking animation for chracters given a system like this.
